# Landing Redesign & Shop Web App

Redesign the frontend landing page with a full Sarko marketing site (keeping mobile app store redirects), and build a new customer web-ordering application as the shop subdomain. The landing includes hero section, features strip, cities list, app download section, and footer. The shop is a Next.js 14 web app with auth, product browsing, cart, coupon support, and order placement with cash on delivery.
