# Sarkis Delivery — Plan Vault

This is the planning vault for **Sarkis Delivery** (national breads, door delivery in Germany, cash on delivery, free shipping).

Each user request gets its own folder containing:
- `request.md` — what was asked
- `plan.md` — the agreed plan
- `changes.md` — log of every change made (written by the `change-logger` agent)
